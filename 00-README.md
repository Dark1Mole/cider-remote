# Cider Remote Pixel – Cursor Build Plan

## Purpose

Build a Cider Remote Android app by using:

- PixelPlayer as the visual/design inspiration
- Cider Remote RN as the functionality/reference base
- Cider v2 API as the main remote-control API

The final app should feel like a modern Material 3 Expressive music player, while controlling a Cider desktop client over the local network.

## Cursor Rule

Do not build everything at once.

Follow the files in this order:

1. `01-repo-strategy.md`
2. `02-api-validation-first.md`
3. `03-project-architecture.md`
4. `04-design-system.md`
5. `05-navigation-and-screens.md`
6. `06-cider-api-services.md`
7. `07-state-management.md`
8. `08-live-updates.md`
9. `09-music-link-handling.md`
10. `10-android-media-controls.md`
11. `11-testing-checklist.md`
12. `12-final-polish.md`

Each step must be completed, tested, and committed before moving to the next.
